# Royal Kludge R75 ISO Wired

Here my research to find the datas for the R75 ISO Wired.

The Firmware `rk851uk_usb_via_v13.hex` is a default one for ISO layout (i have a french keyboard).

The file `r75_uk_usb.json` is the default layout for ISO.

The file `KeyboardCheck.exe` was provide by RoyalKludge to get the information of the keyboard. To use, unplug the keyboard, launch the software, plug the keyboard.
